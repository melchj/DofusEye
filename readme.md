## Running Locally
#### 1. Virtual Environment


start virtual environment:
```bash
$ python3 -m venv venv
```

activate virtual environment:
```bash
$ . venv/bin/activate
```

update dependencies:
```bash
$ pip install -r requirements.txt
```
